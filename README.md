## 주제 선정 및 배경
1.브레인스토밍
- 한국을 방문한 외국인 분석
- 놀이공원 사고 분석
- 노인 교통 사고 분석
- 넷플릭스 및 네이버 영화 분석
2. 선정 이유 및 목표
- 약 10만개의 네이버 영화 사이트 정보 (크롤링을 통한 DB축적)
- 넷플릭스 자료를 이용한 통계
- 영화(줄거리, 제목, 별점, 감독) 검색 기능 제공
